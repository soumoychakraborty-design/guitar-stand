# Guitar Stand

Interactive web app to learn guitar, ukulele, chords, scales, and music theory.

## Features
- Analyze chords and see notes
- Explore inversions
- Discover modes and triads
- About Me modal with contact info

## Installation
```bash
git clone <repo-url>
cd guitar_stand
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
pip install -r requirements.txt
python app.py
